## üöÄ Getting Started

### Prerequisites
- SQL Server or compatible database management system
- SQL Server Management Studio (SSMS) or Azure Data Studio
- Basic understanding of database concepts

### Setup Instructions

1. **Day 01 Setup:**
   - Execute `Day-01-Customer-Order/Database/Customer-Order.sql` to create the database
   - Practice with queries in `Queries/Pract.sql`

2. **Day 02 Setup:**
   - Execute `Day-02-Procedures-and-CRUD/Databse lpuDB/lpuDB.sql` to create lpuDB
   - Work through CRUD operations and stored procedures

3. **Day 03 Setup:**
   - Ensure AdventureWorks sample database is installed
   - Execute queries from the provided files

---

## üìù Learning Path

Follow the days in sequence for a structured learning experience:

1. Start with basic queries and database operations (Day 01)
2. Progress to stored procedures and CRUD operations (Day 02)
3. Master advanced techniques like JOINs and GROUP BY (Day 03)

## üí° Tips for Success

- Compare query results to verify your understanding
